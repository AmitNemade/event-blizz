# EventBlizz (Next.js Pages Router + Tailwind)

This is a full React rewrite of the original multi-page HTML project.

## Stack

- Next.js (Pages Router)
- Tailwind CSS
- Existing project styling preserved via migrated custom CSS

## Routes

- `/`
- `/services`
- `/about`
- `/gallery`
- `/contact`

## Run

```bash
npm install
npm run dev
```

## Build check

```bash
npm run build
```
